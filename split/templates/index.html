About

Split is a web applicaiton that helps you summarize text. Our natural language processing algorithms take your text, identify the key ideas, then return a subset of the sentences that relate closely to those key ideas. Right now we are using a "compression factor" of three i.e., we try to make the input text thrice as short.

We are still in alpha, and this is a bare bones working prototype. If you find a bug of sorts, please use the contact page to let us know. You can also identify issues on the GitHub page for this project.

You can try out (baby?) Split by navigating to the Summarize page.
